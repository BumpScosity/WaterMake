Page2, Functions.

The basics:

`define function-name` Is used to tell the parser where a function starts, and what its name is.

`endef` Is used to tell the parser that a function ended.

`$(call function-name)` Is used to call a function by its name

Example:

_start:

$(call say_something)

_end

define say_something

echo Something

endef
